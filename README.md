# 🌌 Victoria Adenola — Obsidian Hero Landing

A high-impact, minimalist "Power Center" hero section designed for a Product Designer & Full-Stack Developer. This project focuses on modern UI/UX principles, featuring a deep **Obsidian** aesthetic, glassmorphism, and subtle motion design.

## ✨ Key Features

- **Obsidian Dark Mode:** A sophisticated `#0a0a0a` background with vibrant violet (`#8b5cf6`) accents.
- **Glassmorphism:** CTA button featuring backdrop blur and semi-transparent borders.
- **Power Center Layout:** Perfectly centered content for maximum visual focus and conversion.
- **Modern Typography:** High-contrast hierarchy using the Inter font family with tight letter spacing.
- **Glow Accents:** Subtle radial gradient flares to add depth and a premium "Web3" feel.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **Lovable UI:** Inter for a clean, professional sans-serif look.
- **Tailwind CSS:** For rapid, utility-first styling.
- **Google Fonts:** Inter for a clean, professional sans-serif look.
